  2. 随机生成0-10000个随机数并计算花费时间（变相测试电脑性能）：
    import time
import random
number = []
fu = []
dan = []
a = float(time.time())
for num in range(10000):
 number.append(random.randint(0,100000))
while len(number) >0:
 number1 = number.pop()
 if(number1 % 2 == 0):
     fu.append(number1)
 else:
     dan.append(number1)
b = float(time.time())
print(dan)
print(fu)
print(b-a)
  3. 在桌面生产一个具有格式的99乘法表：
    file = open('C://Users/Administrator/Desktop/99.txt','w')
for i in range(1,10):
 j=i
 while j<=9:
     file.write('{} * {} = {} '.format(i,j,i*j))
     j+=1
 file.write('\n')
file.close()
  4. 查找并统计一篇英文短文中每个词汇出现的次数：
    • 先在桌面存放一个test.txt文件并放入一篇文章
    • 再运行查词结果会出现在桌面result.txt文件中
      import string
path = 'C://Users/Administrator/Desktop/test.txt' #可以文章自定义路径
file = open('C://Users/Administrator/Desktop/result.txt','w') #可以自定义输出结果路径
with open(path,'r') as text:
words = [raw_word.strip(string.punctuation).lower() for raw_word in text.read().split()]
words_index = set(words)
counts_dict = {index:words.count(index) for index in words_index}
for word in sorted(counts_dict,key=lambda x: counts_dict[x],reverse=True):
print('{} -- {} times'.format(word,counts_dict[word]))
file.write('{} -- {} times\n'.format(word,counts_dict[word]))
